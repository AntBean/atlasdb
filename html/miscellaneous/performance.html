Performance Benchmarking¶

Running Benchmarks¶

  1. Follow the steps provided in Running from Source.

  2. Build the performance cli:

    ./gradlew atlasdb-perf:installDist
cd atlasdb-perf/build/install/atlasdb-perf/bin

  3. Run the performance cli:

    ./atlasdb-perf --help

Contributing Benchmarks¶

The benchmarks are written using the Java Benchmarking Harness (JMH) bundled with OpenJDK. To get started understanding JMH take a look at the JMH Samples. Our benchmarks are found in the atlasdb-perf project.
